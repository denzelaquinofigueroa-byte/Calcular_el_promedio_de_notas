# # Sistema de Cálculo de Notas

## Descripción

Este proyecto consiste en una aplicación de escritorio desarrollada en **C#** que permite registrar el nombre de un estudiante e ingresar cuatro notas para calcular su promedio académico.

---

## Tecnologías Utilizadas

**Lenguaje:** C#  

**Framework:** Windows Forms (WinForms)  

**Entorno de Desarrollo:** Visual Studio  

---

## Funcionalidades

- Registro del nombre del estudiante.
- Ingreso de cuatro notas.
- Cálculo automático del promedio.
- Validación de notas entre 0 y 100.
- Manejo de excepciones mediante Try-Catch.
- Evaluación de Completivo.
- Evaluación de Extraordinario.
- Visualización del resultado final.

## Uso o Ejecución

1. Abrir el proyecto en Visual Studio.
2. Ejecutar el programa presionando **F5**.
3. Ingresar el nombre del estudiante.
4. Introducir las cuatro notas correspondientes.
5. Presionar el botón **Calcular**.
6. Verificar el resultado mostrado por el sistema.
7. Si el estudiante pasa a completivo o extraordinario, ingresar las nuevas notas cuando el sistema lo solicite.
